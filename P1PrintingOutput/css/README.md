# Hitta din bok

## Beskrivning
En webbapplikation där användare kan söka efter böcker via Open Library API.
Användare kan se bokomslag, titlar, författare och sortera resultaten.

## API
- **Open Library API** (https://openlibrary.org/)
- Ingen API-nyckel krävs
- Hämtar bokdata med titel, författare och omslag

## Funktioner
- Sök efter böcker genom att skriva i sökfältet
- Sortera resultat efter:
  - Titel (A-Ö / Ö-A)
  - Utgivningsår (äldst/nyast först)
- Visar de första 20 resultaten
- Responsiv design med CSS Grid
- Felhantering för tomma sökningar och API-fel

## Tekniker
- HTML5 (semantisk struktur)
- CSS3 (Grid layout, hover-effekter)
- JavaScript (fetch API, dynamisk rendering)
- Open Library API
