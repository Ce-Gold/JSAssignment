# Hitta din bok

## Beskrivning
En enkel webbapp där användare kan söka efter böcker och se information om titel, författare och bokomslag.

## API
Projektet använder Open Library API för att hämta bokdata:
- API: https://openlibrary.org/
- Ingen API-nyckel krävs

## Funktioner
- Sök efter böcker genom att skriva i sökfältet
- Visar de första 20 resultaten
- Visar bokomslag, titel och författare
- Responsiv design

## Tekniker
- HTML5
- CSS3 med Grid
- JavaScript
- Fetch API

## Så här kör du projektet
1. Öppna `index.html` i en webbläsare
2. Skriv ett boknamn i sökfältet
3. Klicka på "Sök" eller tryck Enter
